# DespachoBC - Sitio Web Corporativo

Este es el repositorio del sitio web oficial de **DespachoBC**, una plataforma diseñada para proyectar profesionalismo y claridad en servicios de consultoría y asesoría. El proyecto está construido buscando máxima velocidad de carga (SEO) y una experiencia de usuario limpia.

## 🚀 Tecnologías utilizadas

* **Astro**: Framework web para una entrega de contenido ultra rápida.
* **CSS Grid & Flexbox**: Para un diseño responsivo y simétrico.
* **Componentes Astro**: Arquitectura modular para elementos reutilizables (Cards, Grids, etc.).
* **Diccionarios de Datos**: Separación de contenido y lógica para facilitar el mantenimiento.
